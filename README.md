# 🔗 URL Shortener (Hashing-Based)

## Overview
A C++ console-based URL Shortener inspired by Bitly.

## Features
- Convert long URLs to short codes
- O(1) lookup using unordered_map
- Simulated redirection
- Click count tracking

## Tech Stack
- C++
- Hashing (unordered_map)
- OOP
- Basic System Design

## How to Run

Compile:
g++ main.cpp URLShortener.cpp -o shortener

Run:
./shortener
